HTML

  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user
  • The heading element can be seen with a h1, h2, h3, h4, h5, h6 (h1 being the largest element) and there should only be one h1 on each page
  • The p element can be used for paragraphs
  • The section element can be used to structure the website
  • The br element is used for a new line break
